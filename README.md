# Java Programs

## A Collection of Java OOP Projects

---

## Repository Purpose

This repository is a growing collection of Java programs focused on Object-Oriented Programming (OOP). Each project is designed around a real-world scenario such as delivery systems or ride booking platforms.

The main objective is not just to write functional code, but to build well-structured, maintainable, and concept-driven programs that clearly demonstrate OOP principles in action.

Each project is self-contained and includes its own detailed README file inside the respective folder, explaining:

- System design and structure  
- OOP concepts used  
- Implementation logic  
- How to compile and run  

---

## 01. Delivery System

### Description
A food delivery platform simulation inspired by services like Foodpanda. It manages restaurants, menus, and customer orders across multiple cities.

### OOP Concepts Used
- Composition and Aggregation  
- Deep Copy and Copy Constructors  
- Encapsulation and Defensive Copying  
- ArrayList usage  
- StringBuilder for optimized output  
- Method overriding (`toString`, `equals`)  
- Static ID generation  
- Final fields for immutability  

### Classes
- MenuItem  
- Restaurant  
- Order  
- DeliverySystem  
- Tester  

📁 Full documentation available inside: `DeliverySystem/`

## 02. Ride Booking System

### Description
A ride-hailing simulation system with multiple ride types including Economy and Premium rides. Each ride calculates fare using its own logic.

### OOP Concepts Used
- Inheritance  
- Abstract Classes and Methods  
- Method Overriding  
- Runtime Polymorphism  
- `super` keyword usage  
- Static variables for tracking total rides  
- Encapsulation  
- Final fields for constants  

### Classes
- Ride (Abstract Class)  
- EconomyRide  
- PremiumRide  
- Tester  

📁 Full documentation available inside: `RideBookingSystem/`
